Conventional Neural Networks (CNN)
컴퓨터 비젼(이미지 관련) 부분에 가장 성공적인 뉴런 네트워크 

image classification 에서
object detection 로의 진행

object detection
단순히 물체의 구분 뿐만 아니라 이미지에서 다수의 오브젝트의 위치까지 확인하는
다양한 인식이 가능



실제 학습 과정에서 대상이 되는 이미지는 입력값이 매우 크기에
각각의 값에 대한 w, b 등의 학습이 일반적 방법으로는 매우 힘들다

overfitting
과도하게 정확한 처리를 통해 실제로는 구현하기 힘든 내용으로 인해 문제가 발생하는 상태
파라미터를 과도하게 많이주는 식을 통해 나타남
즉, 이는 위에서 언급한 입력값이 많아 파라미터, 레이어 등이 너무 큰 상태

반대로 underfitting도 존재 ( 너무나 단순하게 처리해 인식하지 못함 )


fully connected == 모든 픽셀이 input 레이어의 node들과 모두 연결됨
local connectivity == 구역별로 노드에 연결됨
conventional == 특정 크기의 필터를 통해 영역을 기준 크기로 변환한 뒤 처리

필터의 처리를 통해 필터마다 필드맵이 생성됨
필터는 하나가 아닌 복수가 존재하며 그 특성이 다를 수 있음


padding
인의적으로 이미지 외부에 픽셀을 추가해 변환 과정에서 목표하는 크기를 맞추게 되는데
이때 여백의 추가 과정을 padding이라 함


stride
필터를 통해 진행하는 convertion에서 필터가 변환하는 간격을 줌으로써 좀더 단순화 시키는 것


결과 CNN은 학습할 파라미터 수가 적어짐으로 학습에서 부담이 더 적어진다



필터의 w, x 는 괜찮지만 depth (채널수) 는 input과 같아야함
output의 경우는 필터의 수 만큼 채널이 생성됨
